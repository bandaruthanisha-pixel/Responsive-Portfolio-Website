# 🌐 Responsive Portfolio Website

A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase my profile, technical skills, projects, and contact information.

---

## 📖 Project Overview

This portfolio website was developed as part of the **InternSpark Frontend Developer Internship**. It presents my academic background, technical expertise, projects, and contact details through a clean, user-friendly, and responsive interface that works seamlessly across desktops, tablets, and mobile devices.

---

## ✨ Features

- 🏠 Home Section
- 👩‍💻 About Me
- 💡 Skills Section
- 📂 Projects Showcase
- 📞 Contact Section
- 📱 Fully Responsive Design
- 🎨 Modern and Clean User Interface
- 🔗 Smooth Navigation
- 🌐 GitHub Pages Deployment

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Install the **Live Server** extension.
4. Open `index.html`.
5. Right-click and select **Open with Live Server**.
